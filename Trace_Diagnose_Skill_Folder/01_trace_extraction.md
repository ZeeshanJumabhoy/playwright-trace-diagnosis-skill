# Phase 1 – Trace Extraction (Using Playwright MCP)

## Objective
Open and analyze `trace.zip` automatically using Playwright MCP.

## Steps
1. Open trace.zip via MCP.
2. Extract:
   - Screenshots
   - Video timeline
   - Console logs
   - Network logs
   - Action timeline
   - Error stack
3. Identify:
   - Last successful action
   - Failing action
   - Associated locator
   - Associated API call

## Output
- Last Successful Action
- Failing Action
- Error Type
- Associated Locator
- Associated API
