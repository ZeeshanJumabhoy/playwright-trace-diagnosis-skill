# Phase 3 – Root Cause Diagnosis

## Objective
Determine the true root cause.

## Decision Tree
1. Locator Issue
   - Outdated selector
   - Dynamic ID
   - Multiple matches
   - Wrong index

2. Flakiness
   - Missing wait
   - Race condition
   - Slow network

3. Application Bug
   - 500 error
   - Incorrect API response
   - Console crash
   - Business logic issue

4. Data Issue
   - Incorrect test data
   - Environment mismatch

## Output
- Failure Category
- Detailed Explanation
- Confidence Level
