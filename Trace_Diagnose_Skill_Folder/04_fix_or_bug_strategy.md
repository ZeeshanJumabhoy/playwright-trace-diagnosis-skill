# Phase 4 – Fix or Bug Strategy

## If E2E or Flaky
- Improve locator strategy
- Add proper waitForResponse
- Use stable selectors (data-testid)
- Adjust assertion timing

Respect framework conventions and naming patterns.

## If Application Bug
Generate structured bug report:
- Bug Title
- Summary
- Repro Steps
- Expected Result
- Actual Result
- Iteration Info
- Reported By
- Assign To

## Output
- Fix Applied OR
- Bug Report Generated
